Bienvenido al Juego. El juego consiste en ir recogiendo todo el dinero posible para comprar objetos de la tienda, una vez que compres todos los objetos ganaras, pero claro no sera tan facil tendras obstaculos que sortear. Para mover al personaje tendras que utilizar la tecla "a" para moverte para la izquierda, la tecla "d" para moverte hacia la derecha y la barra espaciadora para saltar
